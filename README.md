# Salary-prediction-ML-project

This is a simple machine learning project to predict software developer salaries based on different features.

## About the Project
In this project, I used a dataset from Kaggle and built a model to estimate salary using factors like experience, education, country, and technologies used.

## What I Did
- Loaded and explored the dataset
- Checked data using pandas functions
- Plotted salary distribution and correlation
- Converted categorical data into numbers
- Split data into training and testing sets
- Trained a Linear Regression model
- Made salary predictions

## Result
- Model R² score: 0.70
- Experience has a strong impact on salary

## Example Prediction
Predicted Salary: 94924 USD
